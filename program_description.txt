Описание программы для построения японских свечей

Основные компоненты программы

    main.py - Главный исполняемый файл

    load_config(): Загружает параметры из XML-конфига или использует значения по умолчанию

    get_df_from_csv(): Читает CSV-файл, фильтрует данные по дате/времени

    make_candles_df(): Преобразует сделки в свечи OHLC (основная логика)

    save_results(): Сохраняет результаты в CSV-файлы (по одному на инструмент)

    main(): Оркестрирует весь процесс обработки


Порядок обработки данных:

    Загрузка конфигурации

    Чтение входного CSV-файла

    Фильтрация данных по дате/времени

    Группировка сделок в свечи

    Расчет OHLC-параметров

    Сохранение результатов

Параметры запуска

python main.py <input_csv> <output_dir> [--config <xml_file>]

    input_csv: Путь к входному CSV-файлу сделок

    output_dir: Директория для сохранения результатов

    --config: Опциональный путь к XML-конфигу (по умолчанию используются встроенные параметры)
